To compile, simply execute:
    javac *.java
    
To run, execute the Simulator class:
    java Simulator <op0, op1, ...> <input code>
    
    e.g.:
        java Simulator prog.dat
        java Simulator --NC 8 --dump_regs prog.dat
        java Simulator --dump_mem prog.dat
